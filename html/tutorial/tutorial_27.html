스크롤 제어할 시 주의사항

  • 스크롤할때 일반적인 스크롤보다 더 많이 스크롤 되게 하는 소스입니다.
  • js를 새로 만들어서 해당 jquery 소스를 넣고 html에서 호출하면 됩니다.
  • 스크롤을 제어하는 다른 소스에 같이 쓰면 안됩니다.
  • js에 speed값이 크면 한번에 스크롤할때 더 많이 스크롤됩니다.

jquery 셋팅값

var speed = 3000; /* 스크롤을 할때 추가로 스크롤 될 값 (즉, 값이 크면 더 많이 스크롤 됨) */
