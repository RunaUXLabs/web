동영상을 화면에 가득차게

섹션의 특징

  • 동영상을 감싸는 요소 section은 넓이 width고 높이는 jquery에서 $(window).height()와 동일하게 리사이즈됨
  • 화면의 사이즈가 바뀜에 따라 동영상이 맞춰서 최적의 크기로 보임

영상제어

  • 유튜브에서 영상을 퍼오기로 가져오고 난 후에 &autoplay=1&loop=1 추가 (자동 플레이, 반복)
    그리고 &playlist= 를 추가한 후 동영상 주소의 embed/와 ?사이의 영상 코드값을 복사해서 추가함. (영상 반복 플레이)

기본 html 구조

<section class="fullpage">
	동영상 주소
</section>

기본 css 코드

.fullpage{overflow: hidden;} /* 감싸는 영역 section은 overflow:hidden으로 넘치는 영역을 잘라야 함 */
