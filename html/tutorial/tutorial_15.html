배너의 특징

  • 좌우로 이동하는 배너이다. 첫번재 배너과 마지막 배너을 복제하여 배치하는 것이 특징이기때문에 각각의 배너 li를 선택해야 할 경우 nth-child로 선택하면 안되고 반드시 class명을 별도로 줘야 한다.
  • li의 높이값을 계산해서 jquery에서 li를 감싸고 있는 slideBanner class에 준다. (모바일 홈페이지에서 배너의 사이즈를 줄이때 li의 높이값을 조절하면 slideBanner의 높이를 자동으로 조절하도록 함.)
  • li의 넓이는 css에서 조절한다. 모든 li는 넓이가 같아야 한다.
  • ul의 넓이는 jquery에서 li의 갯수를 계산해서 li가 두줄로 되지 않도록 (좌우로 배치되도록) width값을 주고 있다.

기본 html 구조

<div class="slideBanner"> /* obj_wrap 배너의 넓이를 결정(jquery에서 obj_wrap의 넓이값을 계산에서 obj_child에 적용) */ <ul> /* obj_name 배너 전체를 묶어주는 요소로 이 요소가 jquery에서 움직임(반드시 position: absolute;, left: 0; 이어야함) */ <li></li> /* obj_child 배너항목(반드시 float:left;를 줘서 모든 배너이 옆으로 나란히 배치되도록 해야함.) */ </ul> </div>

기본 css 코드

.slideBanner{
    overflow: hidden;
    position: relative;
    width: 100%;
} /* 배너 li의 넓이가 됨 */
.slideBanner ul{
    position: absolute;
    top:0;
    left: 0;
}
.slideBanner ul li{
    float: left;
    height: 배너의 높이;
}
