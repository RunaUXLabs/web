태그 속성명      속성값
  • 여는 태그와 닫는 태그
    대부분의 태그는 여는 태크 <p>와 닫는태그 </p> 한쌍으로 구성되어 있다. 이렇게 닫는 태그를 지정해 주면 정확히 태그가 적용될 범위를 알 수 있다.
  • 대소문자 구분없이 사용가능하나 소문자를 주로 사용한다.
  • 들여쓰기를 하면 문서의 구조를 쉽게 이해할 수 있다.
  • 태그 안에 속성을 정의할 수 있습니다.
    각 태그마다 사용할 수 있는 속성들이 다르고 사용할 수 있는 속성 값도 달라지게 된다. 그리고 속성은 연속해서 여러 개를 함께 사용할 수 있다.
    <a href="../contents/vision.html" target="_blank">VISION 바로가기</a>
  • 태그 중복 문제
    가장 바깥쪽의 여는 태그는 닫는 태그를 가장 바깥쪽에, 가장 안쪽에 있는 여는 태그는 닫는 태그를 안쪽에 넣어야 한다.
    <div><a href="#n"><strong>다음</strong></a></div>
  • 속성 값은 속성에 대한 정보 또는 설정값으로 "(큰 따옴표) 안에 위치 해야 한다.

css 스타일의 특징

자세히보기
h1 { font-size:50px; color:#3060cd; }
선택자   속성      속성값
  • 스타일을 적용할 요소를 적절하게 선택하여야 한다. (선택자)
  • 하나의 요소에 대한 스타일은 { }(중괄호)로 감싸줘야 한다.
  • 여러개의 속성을 동시에 선언 할 수 있다.
  • 속성과 속성값 사이에는 반드시 :(콜론)을 입력해야 하고, 속성값의 끝에는 ;(세미콜론)을 입력해야 그 속성에 대한 정의를 종료할 수 있다.

<body>
	<ul>
		<li>1번</li>

		<li>4번</li>
	</ul>

연결은 css파일 
ul li{width:25%;float:left;}
/* 브라우저 사이즈 640px 이하에서만 적용 */
@media screen and (max-width:640px){
	ul li{width:50%;}/* ul li라는 선택자를 반드시 동일하게 해야함.*/
}
/* 브라우저 사이즈 641px 이상에서만 적용 */
@media screen and (min-width:641px){
	ul li{width:25%;}
}
/* 브라우저 사이즈 450px이하 ~ 320px이상 사이에서만 적용 */
@media screen and (min-width:320px) and (max-width:450px){
	ul li{width:100%;float:none;}
}

반응형 홈페이지의 특징, 미디어쿼리

자세히보기
/* 브라우저 사이즈 모든 사이즈에 적용 */
ul li{width:25%;float:left;}
ul li a{font-size:40px;font-weight:600;color:#666;}

/* 브라우저 사이즈 450px 이하일 경우만 적용 */
@media screen and (max-width:450px){
    ul li{width:auto;float:none;}
    ul li a{font-size:25px;}
}
  • 모든 브라우저 사이즈에서 적용될 css는 모바일에서 적용될 css보다 무조건 상단에 정의되어야 한다. (위에 배치해야 한다)
  • 선택자를 동일하게 해야한다. (선택자가 다르면 스타일이 적용되지 않을 수도 있다.)
  • 브라우저 사이즈 450px 이하에서도 ul li a의 글자색은 color:#666을 상속받아 색을 유지한다.
  • ul li에 적용된 float:left;값을 모바일에서 해제 시키기 위해서 float:none;을 줘야 한다. (모바일에서 값을 유지 하지 않을 것이라면 해제 하는 값을 써줘야 한다.)

서체를 서버에 등록하여 사용하는 방법(나눔고딕 서체 사용법)

  1. eot woff 서체 다운로드
  2. 나눔고딕 서체 eot, woff 파일을 서버에 저장
  3. css에 폰트 정의 : 
    @font-face{
	font-family : NG;
	src : url('../font/NanumGothic.eot');
	src : url('../font/NanumGothic.woff') format('woff');
}
    서체의 경로(url)을 맞춰줘야 한다.
  4. 서체 적용 : font-family: NanumGothic, "나눔고딕", NG;
